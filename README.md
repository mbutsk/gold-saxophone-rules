# золотой стандарт саксофона

помимо `README.md` этот репозиторий содержит
файлы [`gold-saxophone-lyrics.json`](gold-saxophone-lyrics.json) и [`gold-saxophone-rules.md`](gold-saxophone-rules.md)

## че это ваще такое

разработчик любого проекта может добавить в него саксофон например:

* разработчик федор сделал игру в телеграм боте где можно получить
саксофон в инвентарь
* разработчик игнатий сделал бота для учета финансов. в настройках любой пользователь
может включить саксофон
* разработчик сереженька сделал CLI-утилиту для подсчета строк кода
имеющую флаг --saxophone

если саксофон применен, в конце каждого ответа программы
(например сообщения или вывода CLI утилиты)
появляется строка из песни из [`gold-saxaphone-lyrics.json`](gold-saxaphone-lyrics.json)

если ваш кейс не подходит к кейсам выше вы можете добавить саксофон куда хотите

> "лучше саксофон в руках чем журавль в небе" - (с) конфуций

в файле [`gold-saxaphone-rules.md`](gold-saxophone-rules.md)
хранится список золотых правил которыми нужно руководствоваться
при добавлении саксофона в свой проект

> "да прибудет с вами саксофон!" - семен павлович собакин

P.S. [звезданите](https://github.com/mbutsk/gold-saxophone-rules)
пж репозиторий чтобы распространить саксофонство
